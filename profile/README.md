# The Eregion Forge


**مجموعه The Eregion Forge** برای ساخت ابزارهای متن‌بازی شکل گرفته که کارهای مختلف مسیر اپلای تحصیلی را برای دانشجوها ساده‌تر و قابل‌مدیریت‌تر می‌کنند.

این اورگانیزیشن را تعدادی از دانشجویان **دانشگاه تهران** راه‌اندازی کرده‌اند و قرار است به‌مرور پروژه‌های بیشتری به آن اضافه شود و افراد مختلف هم بتوانند در توسعه آن‌ها مشارکت کنند.

اسم **Eregion Forge** از دنیای تالکین الهام گرفته شده؛ جایی که چیزهای مختلف در آن ساخته می‌شوند. اینجا هم قرار است ایده‌هایی که از مشکلات واقعی مسیر اپلای می‌آیند، به ابزارهای قابل استفاده تبدیل شوند.

تمرکز پروژه‌ها روی کارهایی است که معمولا پراکنده، تکراری یا زمان‌بر هستند؛ مثل مدیریت مراحل اپلای، ددلاین‌ها، موقعیت‌ها و ارتباطات.



---
از هر نوع مشارکتی استقبال می‌کنیم؛ توسعه، مستندسازی، پیشنهاد ایده، گزارش مشکل یا معرفی یک نیاز واقعی.
